# The Place Restaurant Landing Page

A responsive restaurant landing page built using HTML5 and SCSS (Sass).  
This project focuses on clean UI design, modular CSS architecture, and responsive layout principles.

---

## 🔥 Features

- Fully responsive design
- Modular SCSS structure (partials + variables)
- Clean semantic HTML
- Modern navigation layout
- Figma-inspired restaurant hero section
- Organized folder structure (portfolio-ready)

---

## 🛠️ Technologies Used

- HTML5
- SCSS (Sass)
- CSS3
- Node.js (for Sass compilation)
